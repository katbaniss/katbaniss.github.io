About the mobile web

What is the mobile web?

The mobile web refers to the world wide web that can be accessed through a web browser installed on a mobile device. Users can access the web either through feature phones or smartphones.

Rise of the mobile web

The rise of the mobile web has its roots in the development of mobile communication engineering. With the introduction of th GSM (Global System for Mobile Communication) network for the first time about 100,000 users were able to gain access with a D-network. The 1995 manufactured Siemens S 3 coupled with the introduction of prepaid tariffs in 1997 and Nokia's first smartphone boosted the network participation. Prices for mobile phones decreased and the mobile phone market with new functionalities boomed. Over the years, additional mobile data services including GPRS, UMTS (General Packet Radio Service) and technologies such as new video functionalities were established and paved the worldwide access to users (see graphic).

Mobile vs desktop users

With the accelerating distribution of smartphones, starting in 2007, the user number of the mobile web started to approach desktop computer users. In early 2014, mobile users exceeded desktop users on the web (see graphic).
